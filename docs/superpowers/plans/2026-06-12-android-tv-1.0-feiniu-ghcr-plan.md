# Android TV 1.0 and Feiniu GHCR Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish Android TV `1.0` and provide public GHCR backend/admin images with a directly deployable Feiniu Docker Compose package.

**Architecture:** Keep the existing NestJS and Vue applications intact, add a narrowly scoped APK download route to the backend, package each application in its own multi-stage image, and use Nginx to proxy admin `/api` calls to the backend service. Feiniu persists the existing backend data and media-cache paths plus a configurable releases directory.

**Tech Stack:** Kotlin/Gradle, NestJS/Node.js, Vue/Vite, Nginx, Docker Buildx, GitHub Actions, Docker Compose.
