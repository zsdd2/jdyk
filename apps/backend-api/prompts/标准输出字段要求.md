photo_analysis.caption 为 100-200 字中文画面描述，直接描述真实可见内容，不要根据文件名或相册名编造。

classification 必须包含 category、scene_tags、tv_suitability；scene_tags 返回 3-6 个中文标签，tv_suitability 只能为 high、medium、low。
